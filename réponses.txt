Quelle est la différence entre la programmation impérative et la programmation déclarative ?
	La programmation impérative implique la description explicite des étapes pour résoudre un problème, tandis que la programmation déclarative se concentre sur la description de ce que le résultat doit être sans nécessairement décrire comment l'obtenir. La programmation déclarative peut être plus abstraite et moins spécifique, mais peut aussi rendre le code plus concis et facile à maintenir.

Qu'est-ce qu'une promise et quelles sont ses utilisations possible ? 
	Une promesse est un objet JavaScript qui représente l'aboutissement ou l'échec d'une opération asynchrone. Une promesse peut être soit "résolue" (c'est-à-dire qu'elle a réussi), soit "rejetée" (c'est-à-dire qu'elle a échoué). 

Comment fonctionne async / await ?
	async/await est une syntaxe de programmation qui facilite l'écriture de code asynchrone en JavaScript. async est utilisé pour déclarer une fonction asynchrone, tandis que await est utilisé pour attendre la résolution d'une promesse dans une fonction asynchrone.
